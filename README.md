#### 2. Usage
iron-cli 설치 이후에는 directory root에서 `$ iron`으로 실행시키시면 meteor가 돌아갑니다. "http://localhost:3000"에서 접속하실 수 있으며 MongoDB는 3001번 포트에서 돌아가도록 설정되어 있습니다.
